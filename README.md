# Installation


1. sudo apt update


2. sudo apt install -y git


3. git clone  https://github.com/by-student-2017/skpar-0.2.4.git


4. mv skpar-0.2.4 skpar


5. cd skpar


6. chmod +x install.sh


7. ./install.sh
